# 🏦 State Bank of India - Banking Management System
![Python](https://img.shields.io/badge/Python-3.12-blue?logo=python)

A modern and user-friendly **Banking Management System** built with **Python** and **Streamlit**. It enables secure account management, banking transactions, customer verification, and PDF statement generation through an intuitive dashboard.

## ✨ Features

- 👤 Create, View, Update & Delete Accounts
- 💰 Deposit, Withdraw & Transfer Money
- 🔍 Search Account & Balance Inquiry
- 📄 Mini Statement & Bank Statement PDF
- 📊 Transaction History
- 🛡️ KYC Verification
- 💳 Credit Card Module
- 💵 Loan Module
- 🏦 Fixed Deposit Module
- 🤖 AI Banking Assistant
